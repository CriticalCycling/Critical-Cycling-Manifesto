クリティカル・サイクリング宣言

　世界は変化している。21世紀を迎えた人類は、利便性に堕して、バランスを逸したモダニズムの終焉にようやくブレーキをかけつつある。リアルな都市は、ヴァーチャルなアーキテクチャーとの接続によって、その存在のあり方を変容しつつある。変形し、生成する時空間を再生するキーワードは、バランスの復権だ。人類最古の発明である車輪にペダルが装着されたのは、19世紀のことである。私たちは、モダニズムの始めに立ち戻り、ハイ・テクノロジーと身体が駆動するバランス感覚に着目し、いま、そのギアを再起動しよう。
　自転車を通じて、車輪と身体は、理性と野生、都市と自然、ヴァーチャルとリアルを接続し、シンプルなバランスの循環を見出す。私たちはこの営みを「クリティカル・サイクリング」と命名し、以下の綱領を宣言する。

1）自転車に乗ることは、楽しみである。
　なぜならヒトは、自分自身の身体を通じて、車輪、ペダル、フレームが構成するグルーブを享受するからだ。つまり自転車は、テクノロジー（技術工学）である。

2）自転車に乗ることは、シンプルである。
　なぜならヒトは、平衡感覚の制御を通じて、機能、習慣、共有に関する感動を瞬時に得るからだ。つまりクリティカルは、エスノロジー（民族誌学）である。

3）自転車に乗ることは、発見的である。
　なぜならヒトは、ペダルの循環運動を通じて、ハンドルとサドルから世界をスキャンしているからだ。つまりサイクリングは、オントロジー（存在論）である。

　愛する者よ、自転車に乗れ！　共に世界を再起動しよう！

2016年7月31日
